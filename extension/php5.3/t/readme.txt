Тесты для модуля sbf
1) Для запуска необъодимо иметь
а) GNUmake http://www.gnu.org/software/make/
b) perl 5.8.7 http://www.1014103.ru/tools/ActivePerl-5.8.7.813-MSWin32-x86-148120.msi
c) php 5.X.X
2) Также необходимо положить тестируемый модуль sbf в папку с тестами
3) запустить команду make

Принцип работы
1) Запускается команда prove
2) Она в свою очередь запускает файлы *.t
3) В этих файлах подключается Suite.pm и из него выполняется пользовательская функция check()
4) В функции check для каждого *.t файла запускается php скрипт test_php.php
5) Этот скрипт формирует sbf файл при помощи php модуля SBF0.php, конвертирует его в текстовый 
формат при помощи модуля php_sbf.dll и сохраняет в файл tst
6) Далее возврат в функцию check и запуск сравнения полученного tst файла c эталонным
7) В результате формируется файл cmp либо с информацией что разницы с эталоном нет, либо описание
имеющихся различий
8) В конце вызывается make clean для очистки временных файлов

Как добавить новый тест на функцию sbf_get_filtered_list:
1) Необходимо добавить файл ХХХ-filter.t следующего содержания
#!perl
use FindBin qw($Bin);
use File::Basename;

use lib "$Bin";
use Suite;

chdir($Bin);
check(basename($0));
2) Добавить файл ХХХ-filter.t.txt. В этом файле допустимы следующие элементы:
а) Массив test описывающий структуру базы
б) Массив fields описывающий структуру полей
в) Массив filter описывающий структуру фильтра
г) элемент eta описывающий количество найденных записей
